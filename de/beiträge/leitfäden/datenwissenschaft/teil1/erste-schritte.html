Zu Beginn des Kapitels erläutert das Buch die Grundlagen der Python-Programmierung und zeigt, wie man die notwendigen Programme und Werkzeuge installiert. Danach wird es uns durch das Erstellen eines Python-Programms namens "hello_world.py" führen.

Einrichten Ihrer Programmierumgebung

Python wird je nach Betriebssystem etwas unterschiedlich installiert. Das Buch erklärt auch, dass es in den kommenden Kapiteln auf die Details eingehen wird.

Python-Versionen

Wie andere Programme entwickelt sich auch Python weiter, wird aktualisiert und leistungsfähiger. Das Buch behauptet, dass 3.7 die neueste Version von Python zum Zeitpunkt des Schreibens ist. Zum Zeitpunkt des Verfassens dieses Blogbeitrags ist die neueste Version von Python 3.12, und 3.13 befindet sich im Vorabveröffentlichungszustand. An diesem Punkt im Buch werden wir feststellen, ob Python bereits auf unserem System installiert ist. Es wurde auch erwähnt, dass es immer noch alte Projekte gibt, die Python 2 verwenden, aber wir müssen Python 3 verwenden.

Die meisten Beispiele im Buch sind kleine, in sich geschlossene Programme, die Sie aus Ihrem Texteditor heraus ausführen, da Sie den größten Teil Ihres Codes im Texteditor schreiben werden. Manchmal werden grundlegende Konzepte in Snippets gezeigt, die durch eine Python-Terminalsitzung ausgeführt werden, um bestimmte Konzepte effizienter zu demonstrieren. Wenn Sie drei spitze Klammern in einer Codeauflistung sehen, betrachten Sie Code und Ausgabe aus einer Terminalsitzung. Wir werden gleich versuchen, im Interpreter auf Ihrem System zu programmieren.

Wir werden auch einen Texteditor verwenden, um ein einfaches Programm namens Hello World! zu erstellen, das zu einem festen Bestandteil des Lernens der Programmierung geworden ist. Es gibt eine lange Tradition in der Programmierwelt, dass das Drucken einer Hello World!-Nachricht auf dem Bildschirm als erstes Programm in einer neuen Sprache Ihnen Glück bringt. Ein so einfaches Programm erfüllt einen sehr realen Zweck. Wenn es korrekt auf Ihrem System ausgeführt wird, sollte jedes Python-Programm, das Sie schreiben, ebenfalls funktionieren.

In Kürze werden wir unser erstes Programm schreiben, das „Hello World!“ ausgibt, eine Tradition unter jungen Programmierern.

Über Integrierte Entwicklungsumgebungen (IDEs)

Unter diesem Titel erzählt uns das Buch vom Sublime Text Editor, aber ich werde Ihnen die IDEs vorstellen, die Sie verwenden können. IDE steht für „Integrierte Entwicklungsumgebung“ und sind die Texteditoren, in denen wir normalerweise Programme entwickeln. Sie verfügen über eine breite Palette von Plugins und ein integriertes Versionskontrollsystem.

PyCharm ist eine integrierte Entwicklungsumgebung (IDE) für Python, die speziell für professionelle Entwickler entwickelt wurde. Es gibt zwei Editionen: Community und Professional. Die Community-Edition ist kostenlos und Open Source, geeignet für reine Python-Entwicklung, während die Professional-Edition erweiterte Funktionen wie Webentwicklung, Datenbankunterstützung und wissenschaftliche Werkzeuge bietet. Auf der anderen Seite ist VS Code ein leichtgewichtiger, Open-Source-Code-Editor von Microsoft, der stark auf Einfachheit und Leistung fokussiert ist. Er unterstützt verschiedene Programmiersprachen durch Erweiterungen und hat eine große Community, die zu seinem Ökosystem beiträgt. Spyder ist eine Open-Source-IDE, die für wissenschaftliches Programmieren in Python entwickelt wurde. Sie integriert wesentliche Pakete wie NumPy, SciPy und Matplotlib für die Datenanalyse und ist daher eine ausgezeichnete Wahl für Wissenschaftler und Ingenieure. Jede dieser IDEs erfüllt unterschiedliche Bedürfnisse und Vorlieben innerhalb der Python-Entwicklergemeinschaft.

Welche IDE Sie verwenden, bleibt ganz Ihnen überlassen. Wenn Sie ein Anfänger sind, empfehle ich VSCode oder PyCharm Community. Erfahrene Programmierer können auch PyCharm Professional oder Spyder verwenden. Wenn Sie den Sublime Text Editor in Betracht ziehen, hier ist die Erklärung aus dem Buch:

Ich werde diese Teile auf die kürzest- und einfachste Weise erklären.

Linux/FreeBSD

Sie können das Paket „python“ über den Paketmanager Ihrer Distribution installieren.



MacOS

Laden Sie die neueste Python-Version von der offiziellen Website herunter und führen Sie den Installationsassistenten aus. Folgen Sie den Anweisungen, und Python wird auf Ihrem System installiert.

Ausführen des Hello World-Programms

Bevor wir unseren ersten Code schreiben, müssen wir eine Datei namens hello_world.py erstellen. hello_world ist unser Dateiname und .py ist die Erweiterung, die anzeigt, dass unsere Datei Python-Code enthält. Nachdem wir die Datei hello_world.py erstellt haben, öffnen wir sie und geben diesen Code ein:

hello_world.py

Speichern Sie die Datei und führen Sie unser erstes Programm aus.

Ausgabe von hello_world.py

Wenn Sie nach dem Ausführen des Programms nichts sehen, könnte etwas schiefgelaufen sein. Überprüfen Sie jedes einzelne Zeichen in der eingegebenen Zeile. Vielleicht haben Sie versehentlich „print“ großgeschrieben oder vergessen, eines oder beide Anführungszeichen oder Klammern hinzuzufügen. Denken Sie daran, dass Programmiersprachen wählerisch bezüglich ihrer Syntax sind, also wenn Sie es nicht richtig machen, werden Sie definitiv Fehler bekommen. Wenn Sie Schwierigkeiten haben, das Programm auszuführen, werfen Sie einen Blick auf die Vorschläge im nächsten Abschnitt.

Fehlerbehebung bei hello_world.py

Wenn Sie Probleme haben, hello_world.py auszuführen, hier einige Lösungen, die Sie ausprobieren können:

  • Überprüfen Sie den Fehlermeldungs-Traceback von Python auf Hinweise, was das Problem verursacht.
  • Machen Sie eine kurze Pause und überprüfen Sie dann sorgfältig den Code auf Syntaxfehler wie fehlende Doppelpunkte oder nicht übereinstimmende Anführungszeichen.
  • Überlegen Sie, ob Sie von vorne beginnen möchten, indem Sie hello_world.py löschen und neu erstellen.
  • Bitten Sie jemanden, die Schritte mit Ihnen zu verfolgen, da diese möglicherweise einen übersprungenen Schritt bemerken.
  • Bitten Sie jemanden, der Python kennt, um Hilfe oder greifen Sie auf die Online-Installationsanweisungen zu.
  • Nutzen Sie Ressourcen wie Foren und Live-Chat-Seiten, um Unterstützung bei Problemen zu erhalten.
